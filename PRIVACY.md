# Privacy Policy

Effective date: April 15, 2026

SmartFileSorter is a local-first macOS product.

## What Data Is Processed

- Selected folder paths.
- Local file metadata needed for sorting (such as file name and extension).
- Local app settings and local undo metadata.

## Where Processing Happens

- Folder analysis and sorting run on the user's device.
- SmartFileSorter does not require a cloud account.

## Data Sharing

- SmartFileSorter does not sell user data.
- SmartFileSorter does not upload file contents to external services by default.

## Local Storage

The app stores settings and recent undo metadata locally to provide continuity across app restarts.

## Contact

For privacy questions, open an issue in this repository.
